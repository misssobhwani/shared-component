# shared-components
React + TypeScript MUI component library shared across all web apps.
Stack: React 18, TypeScript, MUI v5, @mui/x-data-grid-premium, Storybook 8, tsup
Components: PageSpinner, StripedDataGrid, CustomToolbar, CheckboxCell
Convention: each component has its own folder with ComponentName.tsx, ComponentName.stories.tsx, index.ts
Theme: src/theme/theme.ts — replace with actual shared company theme
